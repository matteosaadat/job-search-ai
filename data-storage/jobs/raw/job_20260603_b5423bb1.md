Front-End Developer – Modernization Project
Company: Intertech (client undisclosed)
Location: Remote
Type: Contract (W2, 1099, or C2C)
Rate: $70–$100/hr depending on engagement type

Full job description:
We are seeking a Front-End Developer to join an Intertech-led modernization initiative focused on transforming a large-scale enterprise application. This role will play a key part in rebuilding and modernizing the user interface, replacing legacy UI frameworks with scalable, maintainable, and accessible solutions.

You will collaborate closely with backend engineers and architects as the system transitions to a service-based architecture, supporting hundreds of UI screens across a complex platform.

Primary Responsibilities:
- Develop modern, responsive UI components using React and TypeScript
- Rebuild and modernize approximately 300+ UI screens of medium complexity
- Integrate front-end components with backend services via REST and GraphQL APIs
- Collaborate with backend and architecture teams to align UI with microservices and security patterns
- Build scalable UI architecture using TanStack (Query/Router)
- Implement clean, maintainable styling using Tailwind CSS
- Participate in Agile ceremonies including sprint planning, estimation, and iterative development
- Support accessibility compliance efforts (WCAG 2.0/2.1)
- Work in a collaborative, co-development environment with Intertech teams

Required Skills:
- Strong experience with TypeScript and modern JavaScript (ES6+)
- Hands-on experience with React
- Experience with Tailwind CSS or similar utility-first styling frameworks
- Experience with TanStack (React Query and/or TanStack Router)
- Strong understanding of component-based architecture and modern frontend patterns
- Experience integrating with REST APIs (GraphQL exposure is a plus)
- Proficiency with HTML, CSS, and responsive design
- Experience with Git-based version control
- Ability to interpret and modernize legacy UI workflows

Preferred / Nice-to-Have:
- Experience with GraphQL APIs
- Familiarity with Playwright or other end-to-end testing frameworks
- Experience with modern build tools such as Vite
- Exposure to Docker or containerized environments
- Experience with Angular (for legacy or hybrid environments)
- Experience modernizing UIs for large-scale enterprise systems
- Experience working in public sector or government environments

Notes: High-impact modernization effort, fully remote contract, modern stack (React/TS/Tailwind/TanStack). Rate targeting $70-100/hr W2/1099/C2C.
